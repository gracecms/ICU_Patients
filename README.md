# ICU_Patients
Develop and optimize 3 machine learning models (logistic regression, decision tree and random forest) to predict the survival of each patient,interpret and compare the fine-tuned models.
Includes: Convert data to a tidy format, Export to normalized tables in an SQLite3 database, Use SQL statements (using from within pandas is acceptable) to retrieve the data needed for each visualization.

The dataset is from https://www.kaggle.com/datasets/ukveteran/icu-patients.
